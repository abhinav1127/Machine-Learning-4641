In this project, we used ABAGAIL to implement the optimization algorithms and neural network.
To use ABAGAIL correctly, you must have ant and the latest Java SDK on your computer for it to run correctly.

To run the Traveling Salesman Test, run this command from the directory:
java -cp ABAGAIL.jar opt.test.TravelingSalesmanTest

To run the Count Salesman Test, run this command from the directory:
java -cp ABAGAIL.jar opt.test.CountOnesTest

To run the Credit Card Test (Neural Network), run this command from the directory:
java -cp ABAGAIL.jar opt.test.AbaloneTest
